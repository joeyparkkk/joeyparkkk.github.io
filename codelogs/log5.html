Code Log 5: Images

Images in HTML and CSS

Images are commonly used in websites. They help show information visually and also make the webpage more interesting. In HTML and CSS, images can be added in different ways. Sometimes images are part of the content, and sometimes they are used for design.

HTML image vs CSS background image

Images can be displayed using the HTML <img> tag or by using the CSS property background-image.

The <img> tag is used when the image is part of the actual content of the page. For example, photos, diagrams, or illustrations.

For example: <img src="photo.jpg" alt="example image">

A background image is added using CSS. It is usually used for design or decoration instead of content.

For example: div { background-image: url("background.jpg"); }

The main difference is that the <img> tag creates an image element in the page layout, while background-image places the image behind the content of an element.

Other CSS properties that affect background images

Besides background-image, there are other CSS properties that control how background images appear.

The background-size property controls how large the background image appears.

Example: background-size: cover;

The background-repeat property controls if the image repeats or not.

Example: background-repeat: no-repeat;

Another useful property is background-position, which controls where the image appears in the element.

Example: background-position: center;

Dimensions vs Resolution

Dimensions describe the width and height of an image in pixels, while resolution describes how much detail the image contains within those pixels.
