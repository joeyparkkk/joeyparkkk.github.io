Code Log 1: HTML Semantic Tags

What is the difference between non-semantic and semantic tags in HTML?

Non-semantic tags do not explain the meaning of the content they contain. Tags like <div> and <span> are generic containers that are mainly used for grouping elements or applying styles. When using non-semantic tags, it can be difficult to understand what the content is for just by looking at the HTML structure.

Semantic tags clearly describe the purpose of the content. Tags such as <header>, <nav>, <article>, and <footer> show the role of each part of the webpage. Because of this, the overall structure of the page becomes easier to understand for developers, browsers, and assistive technologies.
