Code Log 3: CSS Media Queries

CSS Media Queries?

A CSS media query lets developers change CSS styles depending on the screen size, orientation, or device type. It works by checking the screen conditions and applying styles only when those conditions are true. Media queries are useful because they help websites look better on different devices like phones, tablets, laptops, and desktops.

width (min-width / max-width)

The width media feature checks how wide the viewport is. It is usually used with min-width or max-width to change styles based on screen size. This is common for responsive layouts.

height (min-height / max-height)

The height feature checks how tall the viewport is. It works like width, but focuses on vertical space. This can be useful for full-screen layouts.

aspect-ratio

The aspect-ratio feature checks the relationship between the width and height of the viewport. It helps adjust layouts for very wide or very tall screens.

orientation

The orientation feature checks if the device is in portrait or landscape mode. This is useful when users rotate their phones or tablets.
