Code Log 4: CSS Color

CSS Color?

In CSS, color is used to make text easier to read and to make the page look organized. Most of the time, I use color for text, background, and borders. Small changes in color can change the mood of the whole webpage.

What do these CSS properties do?

The color property changes the color of text. If I set color: blue;, the text becomes blue.

The background-color property changes the background color behind an element. I can use it on the whole page (body) or just one section.

The border-color property sets the color of a border around an element. It works with border-style and border-width. If the border style is not set, the border might not show.

How does CSS accept color as a value?

CSS can accept color values in different formats.

Named colors are simple words like red, blue, or green.
For example: color: blue;.

Hexadecimal colors start with a hashtag and six characters, like #ff5733. This format represents red, green, and blue values.

RGB colors use numbers to represent red, green, and blue levels.
For example: rgb(255, 0, 0) creates red.

Another format is HSL, which stands for Hue, Saturation, and Lightness.
For example: hsl(200, 70%, 50%).
